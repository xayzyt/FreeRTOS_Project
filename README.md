# FreeRTOS Practice

这个仓库用于记录 FreeRTOS 学习和嵌入式实时系统实践，重点关注任务、调度、同步和资源管理。

## 内容方向

- FreeRTOS 任务创建与调度
- 队列、信号量、互斥锁等同步机制
- 软件定时器和事件组
- 多任务协作与调试记录
- 嵌入式实时系统基础实践

## 学习目标

- 理解 RTOS 和裸机开发的差异
- 掌握 FreeRTOS 常用 API 和设计模式
- 能够将任务拆分、通信和同步应用到实际嵌入式项目中

## 适合展示的能力

- RTOS 基础理解
- C 语言嵌入式开发能力
- 多任务程序结构设计
- 调试实时系统问题的思路

- 增加任务关系图和关键 API 说明
- 整理常见问题和调试经验
